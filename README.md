# Ejemplos de Projectos en React

Espacio con proyectos Base de React Native (0.82) usando la nueva arquitectura tanto en Android como iOS.

Todos los proyectos fueron tomados en base a los cursos online de Fernando Herrera en Udemy


## Inicialización
Cada proyecto debe inicializarse antes de poder correr sus demos (En cada carpeta)

`npm start`

### Inicialización Android
`yarn android`

### Inicialización iOS

`cd ios && pod install && cd ..`

`yarn ios`
